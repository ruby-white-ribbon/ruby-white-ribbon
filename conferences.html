What is this list about?

We believe every conference should have a Code of Conduct and we think information about the presence of a Code of Conduct will be valuable information to both conference attendees and speakers.

The data above is pulled from the great project RubyConferences.org with their permission, and published according to the Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International License.

We re-publish and add onto the data from RubyConferences.org here because we believe an overview of the presence of a Code of Conduct at Ruby conferences is valuable information, but the maintainers of RubyConferences.org have chosen to not add it to their list.

If you find any Ruby conference missing from the list above, please file an issue or submit a pull request to the RubyConferences.org repository. If you find any information about the presence of a Code of Conduct missing or out of date, please file an issue or submit a pull request to the http://ruby-white-ribbon.org repository.
