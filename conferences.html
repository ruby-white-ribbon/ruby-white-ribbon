Conferences

This list intends to provide information about the availability of a Code of Conduct at Ruby conferences, or other approaches taken in order to help provide a safe and comfortable experience to everyone at the conference.

Conferences listed here are not affiliated with the Ruby White Ribbon campaign. This list is a simple collection of information available on conference websites or given via email.

This page is a work in progress.

    What is this list about?

    We believe every conference should have a Code of Conduct and information about the presence of a Code of Conduct will be valuable to conference attendees, speakers and organizers.

    We re-publish and add onto the data from the great project RubyConferences.org with their permission, and added onto/published according to the Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International License.

    If you find any Ruby conference missing from the list above, please file an issue or submit a pull request to the RubyConferences.org repository. If you find any information about the presence of a Code of Conduct missing or out of date, please file an issue or submit a pull request to the http://ruby-white-ribbon.org repository.
